## 소개

이번 글에서는 제가 만든 사이드 프로젝트 **CVFactory**를 소개합니다. CVFactory는 채용 공고와 개인 경력 정보를 입력받아, AI를 활용해 맞춤형 커버레터(자기소개서) 초안을 생성해주는 웹 서비스입니다.

아이디어의 **동기**부터, 작은 프로토타입을 **실서비스 수준으로 끌어올리면서 겪었던 기술적/운영적 난관**까지 함께 다룹니다.

### 1. 동기 — *왜 AI 커버레터 생성기를 만들었을까?*

대부분의 개발자에게는 언젠가 꼭 만들어보고 싶은 “원픽 아이디어”가 하나쯤 있습니다. 제게 그건 “커버레터 작성의 번거로움을 줄이자”였습니다.

1. **반복을 자동화하기** — 구직 과정에서 커버레터를 여러 번 쓰다 보니, 구조가 비슷한 문장을 계속 반복하게 되었습니다. *이 지루함을 기술로 줄일 수 없을까?* 라는 생각이 들었습니다.
2. **새로운 기술을 직접 써보기** — LLM이 빠르게 발전하면서, 채용/문서 작성처럼 실용적인 도메인에 적용해보고 싶었습니다.
3. **풀스택 경험 쌓기** — 프론트엔드부터 백엔드, DB, 배포까지 전 과정을 직접 운영해보고 싶었습니다. Django, Docker, Northflank를 만지기에 딱 좋은 핑계였습니다.

### 2. 기술 스택 & 아키텍처 — *어떻게 만들었나?*

* **Backend**: Django(인증/라우팅) + 내부 FastAPI 마이크로서비스(LLM 추론)
* **Frontend**: Vanilla HTML / CSS / JavaScript
* **DB**: SQLite(dev), PostgreSQL(prod)
* **Deployment**: Docker + Northflank + Cloudflare

사용자가 프론트엔드에서 폼을 작성 → Django가 내부 FastAPI 마이크로서비스를 호출 → 마이크로서비스가 AI 제공자 API를 호출 → 초안을 받아서 Django가 브라우저로 반환합니다. (FastAPI 쪽은 후속 글에서 더 자세히 다룰 예정입니다.) Docker 이미지는 빌드되어 Northflank에서 자동으로 배포됩니다.

### 3. 삽질 포인트 — *무엇이 문제였나?*

#### Django 정적 파일 전쟁
* **문제**: `DEBUG=True`에서는 CSS/JS가 잘 나오는데 `DEBUG=False`에서는 깨졌습니다.
* **해결**: `STATIC_ROOT`, `collectstatic` 개념을 제대로 이해하고, **Whitenoise**를 적용했습니다.

```python
# config/settings.py (snippet)
STATIC_URL = '/static/'
STATIC_ROOT = '/app/staticfiles/'

STATICFILES_DIRS = [
    BASE_DIR,
]

STATICFILES_STORAGE = 'whitenoise.storage.CompressedManifestStaticFilesStorage'
```

#### Docker 이미지 비대화
* **문제**: 첫 Docker 이미지가 수백 MB까지 커졌습니다.
* **해결**: `.dockerignore`를 추가하고, `python:3.8-slim-buster` 같은 슬림 베이스로 변경, 멀티 스테이지 빌드를 적용했습니다.

```dockerfile
# Dockerfile (snippet)
FROM python:3.11-slim AS builder

WORKDIR /app
COPY . /app

# install deps fast using uv
RUN apt-get update && apt-get install -y curl ; \
    curl -LsSf https://astral.sh/uv/install.sh | sh ; \
    uv pip install --no-cache-dir --system -r requirements.txt
```

#### 새로운 PaaS: Northflank
* **문제**: Render의 무료 플랜은 콜드 스타트 중심이라, 상시 실행(핫 스타트)에 가까운 무료 PaaS가 필요했습니다.
* **해결**: 상시 실행을 제공하는 드문 무료 PaaS 중 하나인 **Northflank**로 이동했고, GitHub/Docker 연동을 구성했습니다. 배포 시 CDN 캐시를 비우기 위해 `northflank.json`과 `purge_cloudflare_cache.py`도 만들었습니다.

```python
# purge_cloudflare_cache.py (snippet)
import os, requests, logging

logging.basicConfig(level=logging.INFO)

def purge_cache():
    api_token = os.environ.get("CLOUDFLARE_API_TOKEN")
    zone_id  = os.environ.get("CLOUDFLARE_ZONE_ID")
    requests.post(
        f"https://api.cloudflare.com/client/v4/zones/{zone_id}/purge_cache",
        headers={"Authorization": f"Bearer {api_token}"},
        json={"purge_everything": True},
        timeout=30,
    )
```
