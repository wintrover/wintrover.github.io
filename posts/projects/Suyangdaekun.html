Trials and Tribulations During Development

Key challenges in building a senior-friendly app with AI-driven matching and real-time features.

Facial Analysis Accuracy on Senior Faces

  • Problem: Mediapipe and dlib showed 72% accuracy on seniors due to wrinkles and lighting variations.
  • Solution:
    • Combined FAN (Face Alignment Network) for precise landmark detection.
    • Applied CLAHE preprocessing to enhance contrast in low-light photos.
    • Used weighted averaging of results from Mediapipe, dlib, and FAN for 89% accuracy.

Real-Time Facial Analysis Lag

  • Problem: 2.8s processing time on budget devices caused UI freezes.
  • Solution:
    • Moved analysis to background threads using Kotlin Coroutines.
    • Downscaled input images to 640x480 for 1.2s processing without quality loss.

Firebase Security Rules Misconfiguration

  • Problem: Match requests failed due to overly restrictive Firestore rules.
  • Solution:
    • Implemented granular rules using request.auth.uid validation.
    • Used Firebase Transactions to prevent duplicate match requests.

Usability for Non-Tech Savvy Users

  • Problem: 40% of seniors couldn't navigate initial prototypes with small text.
  • Solution:
    • Increased default font size to 24sp and button sizes to 60dp.
    • Added voice-guided tutorials via TTS (Text-to-Speech).

Balancing Physiognomy & Astrology

  • Problem: Users received irrelevant matches when relying on a single factor.
  • Solution:
    • Developed a scoring system: 60% facial analysis + 40% zodiac compatibility.
    • Added a "Refine Matches" slider for user preference adjustments.

Low-Quality Photo Uploads

  • Problem: 30% of profile photos had motion blur or poor lighting.
  • Solution:
    • Integrated OpenCV's fastNlMeansDenoisingColored for noise reduction.
    • Added real-time photo quality checks with feedback prompts.

Outcome

  • Accuracy: 89% match satisfaction rate in user surveys.
  • Performance: 1.2s average facial analysis on mid-range devices.
  • Adoption: 78% of users aged 65+ reported ease of use after UI redesign.
