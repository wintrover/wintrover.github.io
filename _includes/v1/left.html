개발 기간: 진행 중

사용 기술: Python, Django, FastAPI, Celery, Playwright, BeautifulSoup, Langchain (Groq API, Cohere Embeddings, FAISS), HTML, CSS, JavaScript, Docker, Docker Compose, GitHub Actions, Google Cloud Run (Backend), Northflank (Frontend), Upstash Redis, SQLite/PostgreSQL, Gunicorn, Whitenoise, Cloudflare

주요 기능 및 도전 과제:

  • Backend (CVFactory Server):
    • Playwright와 BeautifulSoup를 이용한 웹 페이지 정보 비동기 스크래핑 및 HTML 파싱 (iframe 재귀 탐색 포함).
    • FastAPI 기반 백엔드 API 및 Celery, Upstash Redis를 이용한 비동기 작업 처리 시스템 구축.
    • Langchain 라이브러리를 활용한 RAG(Retrieval Augmented Generation) 파이프라인 구축:
      • Cohere 임베딩 및 FAISS 벡터 저장소를 사용한 시맨틱 검색으로 관련 정보 추출.
      • Groq API (LLM) 연동을 통한 문맥 기반 맞춤형 자기소개서 생성.
  • Frontend (CVFactory Web):
    • Django 프레임워크 기반 사용자 인터페이스 개발.
    • 채용 공고 URL 및 사용자 스토리 입력을 통한 자기소개서 생성 요청 기능.
    • 생성된 자기소개서 확인 및 수정 기능 제공.
  • 공통 및 인프라:
    • Docker, Docker Compose를 활용한 백엔드 및 프론트엔드 애플리케이션 컨테이너화.
    • GitHub Actions를 이용한 CI/CD 파이프라인 구축:
      • Backend: Google Cloud Run 자동 배포 (GCP Secret Manager 연동).
      • Frontend: Northflank 자동 배포 (Cloudflare 캐싱 연동).

향후 개선: AI 콘텐츠 추천, 키워드 최적화, 채용 플랫폼 연동.
